# Balatro 更好的简体中文本地化

## 安装

### Steamodded

如果你已经安装 [Steamodded](https://github.com/Steamopollys/Steamodded)，将 `Better_zh_CN` 整个文件夹放置在 `Mod` 文件夹下即可。

### 其他方式

仓库中的 `zh_CN.lua` 文件格式与官方同名文件一致，你可以自行寻找其它方式载入该文件。
